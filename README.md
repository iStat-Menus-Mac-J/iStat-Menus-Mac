## Overview

iStat Menus Professional Suite is a complete system monitoring and diagnostics platform that serves power users and IT professionals with the sensor depth, alert infrastructure, and historical data needed for proactive system health management and performance optimization. It goes beyond passive monitoring into active system health management with alerts, notifications, and trend analysis that surface problems before they affect work. The application is designed to run continuously, providing an always-available performance reference that becomes part of the working environment.

Weather integration displays current conditions, hourly and daily forecasts, and weather maps directly from the menu bar alongside system metrics, providing environmental context alongside hardware data in a single persistent interface. Weather data includes temperature, humidity, wind, precipitation probability, and UV index with configurable location. The weather module supports multiple saved locations for users who travel or monitor conditions at multiple sites.

Custom notifications fire when any monitored metric crosses configurable thresholds — CPU temperature exceeding safe limits, available storage dropping below a minimum, network upload spikes suggesting unexpected data transfer, or battery health declining beyond normal parameters. Notifications are delivered through the system notification center with configurable severity levels and repeat intervals. Custom notification rules can combine multiple conditions, alerting only when specific combinations of metrics indicate a genuine problem.

Sensor data includes temperature readings from every available thermal sensor, fan speeds with actual versus target RPM comparison, voltage readings from power rails, and battery health metrics including cycle count, capacity degradation, and charge rate. The sensor display covers CPU, GPU, memory controller, SSD, ambient, and other sensors that the hardware reports. Historical sensor data builds a record of thermal behavior over time, enabling identification of trends that indicate hardware changes or cooling system degradation.

iStat Menus Professional Suite is the system monitoring platform for users who consider hardware awareness a fundamental part of responsible system ownership and want the depth of monitoring data, alerting capability, and historical analysis that informed performance management requires. Its permanent menu bar presence means that system health information is always exactly one glance away.

## Performance & Integration

iStat Menus is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while iStat Menus processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
